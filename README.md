# sao-specless-template

> Generates a Specless boilerplate creative template

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### Run

```bash
sao specless/creative-template my-project
```
